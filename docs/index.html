About

Purpose and target audience

The purpose of this website is to visualize data from row2k to help the NEIRA seeding committee do their job. At this time it is specifically intended for use by the NEIRA Girls Fours seeding committee, but if there is interest it can be expanded to support other NEIRA committees, coaches, and rowers.

Data sources

All data presented here comes from https://www.row2k.com/results/index.cfm?league=NEIRA&year=2024. Any mistakes found in the data should be corrected on the row2k website, since that is the authoritative source of information.

If you find that row2k is correct, but this site is wrong, please let us know by filing an issue.
